> **Purpose**: This document serves as a comprehensive TODO list and implementation guide for Payload CMS integration in a multi-vendor e-commerce marketplace built with Next.js, Payload CMS, tRPC, and Stripe.
>
> **For LLMs**: This file contains detailed task breakdowns, technical implementation details, and code references. Each task includes completion status, technical details, and file paths. Use this document to understand how to use Payload CMS in this project and implement new features following the established patterns.

## System Overview

**What is Payload CMS?**
- **Database Layer** - MongoDB integration via Mongoose adapter
- **Admin Panel** - Auto-generated admin UI at `/admin`
- **Type Safety** - Auto-generated TypeScript types
- **Access Control** - Built-in authentication and authorization
- **File Uploads** - Media management with image processing
- **REST & GraphQL APIs** - Auto-generated API endpoints (not used in this project)
- **Local API** - Direct database access in server-side code

**In this project, Payload serves as:**
- Backend database for products, categories, users, orders, vendors, customers
- Admin panel for content management
- Type-safe data layer via auto-generated TypeScript types

**Key Concepts:**
- **tRPC Integration**: Payload injected into tRPC context as `ctx.db` for client-side operations
- **Direct Payload API**: Used in server components, webhooks, seed scripts for direct database access
- **Access Control**: Collection-level access rules enforced by Payload
- **Bypass Access Control**: Use `payload.db.*` methods only in webhooks, seed scripts, background jobs
- **Type Generation**: Run `npm run generate:types` to generate TypeScript types after collection changes

**File Structure:**
- Payload config: `src/payload.config.ts`
- Collections: `src/collections/*` (Users, Products, Orders, Vendors, Customers, Categories, etc.)
- Generated types: `src/payload-types.ts` (auto-generated)
- tRPC init: `src/trpc/init.ts` (injects Payload into context)
- Admin panel: `/admin` (auto-generated by Payload)
